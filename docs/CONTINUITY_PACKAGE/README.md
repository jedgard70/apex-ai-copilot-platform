# Continuity Package Index

This folder is the compact handoff package for Apex AI Copilot.

Files:

- `APEX_AI_COPILOT_CONTINUITY.md` - product truth, safety rules, architecture, important files.
- `APEX_AI_COPILOT_CURRENT_STATE.md` - current committed state, what works, what does not work yet.
- `APEX_AI_COPILOT_NEXT_STEPS.md` - exact continuation order and validation checklist.
- `APEX_AI_COPILOT_NEW_CHAT_PROMPT.md` - short prompt to paste into a new chat.

Recommended use:

1. Keep this folder updated after major checkpoints.
2. If a chat becomes slow, paste the new chat prompt into a fresh session.
3. Ask the new assistant to read these files before editing code.
