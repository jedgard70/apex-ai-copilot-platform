## Objetivo
Criar artefatos web mais complexos (React/Tailwind/shadcn) e exportar em HTML unico para entrega.

## Fluxo base
1. Inicializar projeto do artefato
2. Implementar interface e logica
3. Empacotar em `bundle.html`
4. Entregar para visualizacao

## Quando usar
- Quando o artefato precisa de estado, componentes reutilizaveis e interacao mais rica.
- Evitar para casos simples que cabem em HTML unico direto.

## Qualidade
- Evitar visual generico “AI slop”
- Padrao consistente de UX
- Entrega final autocontida
