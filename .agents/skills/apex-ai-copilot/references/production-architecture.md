## Clean Architecture Target
- Copilot-first shell: chat, upload, preview/result, and module tools.
- Server-side AI runtime with OpenAI-compatible chat and vision flow.
- File intake separates upload acceptance from deep analysis capability.
- Memory index should route compact repository/local knowledge by domain.
- Viewer/converter layers must be explicit: image preview, PDF extraction/preview, IFC viewer, RVT/CAD conversion path.

## Rebuild Cleanly
- Build from current production truth rather than PR-by-PR historical patches.
- Keep old audit/PR docs as provenance only.
- Extract patterns from code, but rewrite where old modules are patchy or too coupled.
