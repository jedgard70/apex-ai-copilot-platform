# Integração de Prompts de IA para Humanização de Plantas

## Prompts para DALL-E / GPT-4V
- **Planta Residencial**: "Visualização fotorealista de um apartamento moderno de 400 sqft com duas pessoas..."
- **Espaço Comercial**: "Renderização arquitetônica aérea de um escritório aberto..."
- **Cafeteria**: "Renderização profissional do plano de um café..."

## Parâmetros do Midjourney
- Aspect Ratio: --ar 4:3
- Estilo: --style raw
- Qualidade: --quality 2

## Recursos para Renderização 3D
- Bibliotecas gratuitas de figuras humanas como Sketchfab, CGTrader, TurboSquid.
- Dicas para configuração no Blender para uma renderização eficaz.
