# UI / UX Patterns

## Current Product Direction
- White, premium, spacious Copilot-first workspace.
- The uploaded file/preview/result should be visible.
- Main intelligence must be chat conversation, not objective cards.
- Quick chips/buttons are secondary continuations.
- Owner/dashboard controls should not dominate first user experience.

## Reuse/Rewrite
- Reuse visual identity, wide cockpit direction and clear EN/PT handling.
- Rewrite old admin-like or card-first screens.
