# Platform Roadmap

## Recommended Clean Rebuild Sequence
1. Copilot runtime and behavior: real conversation, skill routing, compact memory, safety, auth-aware file analysis.
2. File intake and preview: universal upload, image vision, PDF extraction/fallback, clear unknown-file route.
3. ArchVis/Humanizacao: image/planta analysis, humanized plan prompts, render package workflow.
4. BIM/3D viewer: real IFC viewer, honest RVT/CAD conversion strategy, no fake model views.
5. DirectCut/video: script/shot-list/timeline tool called by Copilot.
6. Project workspace persistence: save intake, files, outputs, decisions and module context.
7. Budget/contracts/field operations: add operational modules after the core Copilot + viewer/file layer is stable.
