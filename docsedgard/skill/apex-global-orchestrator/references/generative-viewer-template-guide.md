# Generative Viewer Template Guide (Finalizado)

Fonte: `D:\AI Jedgard\skill\viewerl.txt`

## Objetivo
Padrao de viewer interativo para arte generativa com:
- navegacao por seed
- controles de parametros
- controles de cor (opcional)
- acoes de reset/regeneracao

## Estrutura recomendada
1. Sidebar de controle
2. Area de canvas principal
3. Script p5.js com parametros centralizados

## Regras
- Seed sempre reproduzivel
- Controles devem atualizar visual em tempo real
- Interface limpa e responsiva
- Separar claramente partes fixas da UI e partes variaveis do algoritmo
