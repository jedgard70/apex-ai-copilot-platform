# DirectCut / Video Intelligence

Use for construction videos, timelapse, walkthroughs, tours, marketing clips, social reels, field evidence, before/after, and render-to-video workflows.

## Analyze Input
- If image: propose camera movement, scene sequence, text overlays.
- If video: propose cut structure, shot list, issues, captions, pacing.
- If BIM/render: propose walkthrough/tour/timelapse path.
- If field photos: propose progress report or issue narrative.

## Output
Produce a video plan:
- purpose: sales, progress, investor, social, technical, training
- duration
- scenes/shots
- voiceover or captions
- transitions
- music/style
- deliverables: reel, YouTube video, client presentation, site report
