# Field Operations / RDO / Site Intelligence

Use for site photos, RDO notes, progress reports, punch lists, safety issues, material deliveries, crew records, quality inspections, and execution problems.

## Analyze
- location/context
- discipline/trade
- progress evidence
- risk/quality/safety issue
- responsible party if known
- date/weather/crew/material if available

## Output
- field record summary
- issue severity
- missing evidence
- recommended action
- RDO/punch-list language
