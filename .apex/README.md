# 🧠 Apex AI Copilot - Core System

Esta pasta secreta (`.apex`) é o coração do licenciamento e das configurações globais da **Apex AI Copilot Platform**.

## Por que esta pasta existe?
Como o sistema que estamos criando não é apenas um site, mas sim um **Motor de Inteligência Artificial** (SaaS - Software as a Service), essa pasta permite que futuramente você:

1. Empacote a API e cobre de outras empresas para instalarem o seu motor.
2. Defina os limites (rate limits) e chaves de licença (Enterprise, Startup, etc) em `config.json`.
3. Separe o que é regra de negócios da interface gráfica.

Tudo que dita o "comportamento corporativo e de venda" do seu ecossistema vai residir aqui dentro!
