## 1. Introduction & Context
With the proliferation of AI and Model Context Protocol (MCP) servers, real-time BIM model synchronization and automated validation have become major paradigms in construction technology. This document outlines research criteria for the offshore-BIM-production-to-US-permits pipeline.

## 2. Research Focus Area
- Automated Code Compliance (IBC/IRC)
- Relational schema designs for BIM-linked databases
- Cross-platform model sharing (IFC/GLB to Autodesk Revit)
- Multi-agent cognitive planners for risk reduction

## 3. Recommended Frameworks
1. **Model Context Protocol (MCP)**: Connecting LLMs to Revit Python instances.
2. **IfcOpenShell**: Validating geometry and data structure on the server.
3. **Autodesk Platform Services (APS)**: Cloud visual rendering.
