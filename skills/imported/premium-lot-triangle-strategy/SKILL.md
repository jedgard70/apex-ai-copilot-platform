# Premium Triangle Lot Strategy

Use this skill when the user asks about triangular lots, irregular site strategy, premium residential concept positioning, lot visualization, or neon/diagram prompt generation.

## Workflow
- Capture lot geometry, access, sun/wind clues, slope/topography, setbacks, views, privacy, target buyer, and program.
- Treat any zoning, legal, environmental, or permit statement as NEEDS VERIFICATION unless sourced by the user.
- Convert the lot into design opportunities: arrival, privacy edge, view axis, service zone, social core, garden/pool logic, and facade moment.
- Use the JSON references as prompt/workflow patterns for lot identification and neon visualization, not as proof of actual site data.
- Produce strategy, concept, prompt, or checklist depending on the request.

## Output Patterns
- Triangle lot concept strategy.
- Premium residential positioning note.
- Lot analysis checklist.
- Neon visualization prompt.
- Site constraints and unknowns table.
