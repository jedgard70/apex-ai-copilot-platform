# Topography And Landscape Visualization

Use this skill when the user asks for topography-aware prompts, terrain visualization, landscape concepts, hologram-style site visuals, or masterplan overlays.

## Workflow
- Identify site facts: slope, access, contour lines, vegetation, water, view, neighbors, and project type.
- Do not invent survey data, legal boundaries, elevation numbers, soil data, or drainage conditions.
- Produce visual prompts that distinguish terrain, built mass, paths, landscape, lighting, and annotation style.
- For technical outputs, label unknown site data clearly and request survey/topography files if precision is needed.
- Keep creative hologram visuals separate from technical survey conclusions.

## Output Patterns
- Topographic hologram prompt.
- Landscape visualization prompt.
- Site analysis checklist.
- Masterplan overlay brief.
- Terrain constraints and assumptions list.
