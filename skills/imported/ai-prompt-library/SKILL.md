# AI Prompt Library

Use this skill when the user asks for prompt ideas, prompt packs, reusable prompt structures, or adaptation of generic prompt collections into Apex workflows.

## Workflow
- Ask only for missing essentials: objective, audience, domain, output type, and constraints.
- Select or compose prompts by use case instead of dumping a generic list.
- Adapt prompts into the user's language and project context.
- Mark broad library content as reference until source material is inspected or user validates the exact desired direction.
- Avoid prompts that request secrets, policy bypass, credential extraction, or unsupported claims.

## Output Patterns
- Curated prompt pack.
- Domain-specific prompt chain.
- Prompt improvement pass.
- Negative prompt and quality checklist.
- Prompt-to-skill conversion outline.
