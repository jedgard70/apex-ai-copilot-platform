# Architecture And Interior Design Prompts

Use this skill when the user asks for prompts for architecture, interiors, facades, concept boards, architectural composition, or design presentation.

## Workflow
- Identify the design category: facade, interior, plan, landscape, product, detail, or mood board.
- Convert the request into a professional prompt with style, spatial intent, materials, lighting, camera, rendering engine style, and constraints.
- Include negative prompts for geometry changes, distorted perspective, poor scale, fake labels, extra rooms, and inconsistent materials.
- Provide multiple versions when useful: conservative, premium, futuristic, minimal, or commercial.
- Keep claims about codes, approvals, and construction feasibility labeled as unverified unless sourced.

## Output Patterns
- Architecture prompt pack.
- Interior design prompt pack.
- Facade prompt.
- Mood board prompt.
- Concept narrative.
