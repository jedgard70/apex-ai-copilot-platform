# Futuristic Interior Mood Board

Use this skill when the user asks for interior mood boards, futuristic room concepts, material palettes, lighting direction, or image prompts for interiors.

## Workflow
- Identify room type, target audience, budget signal, desired mood, must-have materials, and restrictions.
- Build a mood board with palette, materials, lighting, furniture language, textures, art/decor level, and camera angle.
- For futuristic interiors, prefer controlled palettes, integrated lighting, clean geometry, high-quality materials, and restrained visual density.
- Separate concept suggestions from confirmed project requirements.
- Return a checklist when the user is preparing execution or client review.

## Output Patterns
- Futuristic interior prompt.
- Mood board specification.
- Material and lighting palette.
- Room-by-room checklist.
- Client presentation copy.
