# Universal Construction File Intake

Accept every file. Interpret progressively.

## Evidence Levels
1. Metadata only: filename, extension, MIME, size.
2. Preview visible: image/PDF/video/model shown but not deeply parsed.
3. Text extracted: document text, tables, CSV rows, OCR if necessary.
4. Model parsed: IFC entities, object tree, materials, spatial hierarchy, quantities.
5. Workflow-ready: enough facts to produce deliverable.

Always state the evidence level before analysis.

## Routing Table
- Images: ArchVis, humanized plan, marketing, field evidence.
- PDF/DOC/DOCX: contracts, proposals, reports, permits, memorials, budget docs.
- XLS/XLSX/CSV: budget, quantities, procurement, schedule, CRM.
- IFC: BIM/3D/viewer, clash, quantities, coordination, ArchVis prep.
- RVT: conversion/import required; route to BIM import pipeline.
- DWG/DXF/DWF/DWFX/SKP: CAD conversion/viewer required; route to BIM/CAD intake.
- FBX/OBJ/STL/GLB/GLTF: 3D viewer/render/visual model review.
- MP4/MOV/AVI: DirectCut, timelapse, tour, field evidence, social video.
- Unknown: accept, ask objective, offer technical review / document review / visual path.
