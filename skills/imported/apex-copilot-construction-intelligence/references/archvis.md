# ArchVis / Humanização Intelligence

Use for floor plans, facade images, interiors, render requests, moodboards, real estate visuals, marketing assets, and BIM models intended for visualization.

## Analyze Images
Identify visible facts:
- project type: house, apartment, commercial, office, mixed use
- view type: floor plan, elevation, facade, interior, site plan, render
- spaces/zones: social, private, service, exterior, circulation
- style indicators: modern, luxury, minimal, rustic, industrial, tropical
- daylight, landscape, pool, deck, furniture, access, parking
- constraints: missing scale, resolution, annotations, north, dimensions

## Output
Give a conversational analysis and produce a render/humanization brief:
- camera/view
- target style
- materials
- furniture/people/landscape
- lighting
- deliverable: still render, plan humanization, portfolio image, social post, sales package
