# Apex AI Copilot Advanced

## Distinction
- Apex Global Orchestrator skill: owner-level ChatGPT login assistant for Jose Edgard, with full strategic command.
- Apex AI Copilot Advanced: internal platform Help AI, floating on all pages, permission-scoped by user role.

## Current state
The platform has a global floating help/copilot foundation: ApexCopilot present in authenticated routes. It is a foundation, not the full advanced copilot yet.

## Target capability
The internal Copilot should:
- read official `.md` knowledge base
- consult Mission Control
- explain current package status
- answer which module is ready/pending
- consult CRM, projects, proposals, services, documents, agents, events
- generate Codex/Claude handoff drafts for owner/admin roles
- explain platform features as a living manual
- answer general questions: engineering, math, languages, business, documentation
- enforce permissions by role

## Roadmap suggestion
Implement as a later package, for example `PACOTE MASTER 003 — Apex AI Copilot Advanced`, after the core commercial and UX data layers are stable enough for the Copilot to query real state.
