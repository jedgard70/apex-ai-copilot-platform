# Processo de contribuição
**Status:** Foundation | **Versão:** 0.1 | **Data:** 2026-07-20

Issue define problema e domínio. Mudança estrutural cria ADR. Branch/commit/PR seguirão política Git futura. PR declara impacto em Core/produtos, segurança, custo, migração, testes, docs e rollback. Review exige owner do domínio e segurança quando aplicável. Merge/deploy somente após gates e autorização definidos.
