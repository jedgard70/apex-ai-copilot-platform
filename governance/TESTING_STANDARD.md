# Padrão de testes
**Status:** Foundation | **Versão:** 0.1 | **Data:** 2026-07-20

Risco determina testes: unitário para regras, contrato para fronteiras, integração para adapters, E2E para jornadas críticas, segurança para autorização/tenancy e avaliação para IA. Testes são determinísticos ou registram tolerância. LIVE requer evidência repetível. Dados sensíveis não entram em fixtures. Flaky tests são defeitos.
