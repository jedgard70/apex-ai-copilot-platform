# Padrão de documentação
**Status:** Foundation | **Versão:** 0.1 | **Data:** 2026-07-20

Documento contém título, status, versão, data, owner quando definido, estado atual versus proposta, riscos e links relativos. Também declara ou herda classificação oficial: Vision, Constitution, Architecture, Business, Products, Knowledge ou Implementation, conforme [ADR-0018](../architecture/adr/ADR-0018-KNOWLEDGE-IS-A-CORPORATE-ASSET.md). Alegações operacionais citam evidência. Mudanças de contrato atualizam documentação no mesmo ciclo. Docs obsoletos são marcados, não silenciosamente contraditos.
